How to Add Content
1
Upload the file to Google Drive
Right-click the file → Share → set to Anyone with the link → Viewer. Copy the link.
2
Open the Content Sheet
Add a new row. Fill in Subject, Chapter, Topic, Content Type, Title, and paste the Drive link into file link or id.
3
For YouTube videos
Paste the YouTube URL directly into file link or id. Set Content Type to youtube. No Drive upload needed.
4
App updates automatically
The student app re-reads the sheet on every launch. Changes appear within 60 seconds — no rebuild needed.
5
Add a chapter summary (optional)
Open the Metadata Sheet and add a row with Subject, Chapter, Summary, and Detailed Summary. This appears at the top of the chapter in the student app.
