# Parent workflow for a realistic LearnNest pilot

## Best workflow

### On computer
- Upload lesson files to Google Drive
- Organize them under:
  - `LearnNest Pilot / Subject / Chapter / Topic`
- Maintain metadata in Google Sheets

### On mobile
- Avoid CSV upload inside the app
- If needed, parent can still edit the Google Sheet from mobile browser or Google Sheets app

## Why this is realistic

A student app should be simple and stable.
Administrative tasks like content uploads and metadata editing are easier on desktop and can still be done from phone through Google services.

## Suggested split

- **Parent admin:** desktop web page + Google Sheet
- **Student app:** read-only learning experience

## Metadata source options

1. **Best:** Google Sheet
2. **Okay for backup:** CSV stored in Google Drive
3. **Not ideal for mobile:** uploading CSV from inside the app
